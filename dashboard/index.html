How to Run Locally

Start PostgreSQL and Redis, then run the Spring Boot application from the executable module.

docker compose up -d
mvn -pl ecommerce-app -am spring-boot:run
mvn clean verify

What this dashboard is

A static portfolio page published through GitHub Pages. It summarizes the project and links to generated artifacts.

What this dashboard is not

It does not run Spring Boot, PostgreSQL, Redis, workers, or backend services. Live backend execution belongs on a real hosting platform.
